# Курсовая № 4. Парсинг сайтов HH и SJ

## Что позволяет делать программа:

- Выбирать сайт для парсинга
- Выбирать ключевое слово для поиска
- Сортировать вакансии по зарплате

## Классы для работы:

`GetError`- Класс исключения получения данных

'DeleteError' - Класс исключения удаления

`FileSaver` - Абстрактный класс для работы с вакансиями
'JSONSaver' - Класс для работы с вакансиями в формате JSON
